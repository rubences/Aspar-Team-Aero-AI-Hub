# Aspar Team Aero AI Hub

Digital pit wall and AI-assisted aerodynamic analysis platform.

## Core Pillars
- **Babai Quantization**: High-efficiency telemetry compression.
- **GRU Predictors**: Real-time dynamics and anomaly detection.
- **Edge-RAG**: Microsecond retrieval of historical context.
- **Human-in-the-Loop**: Operational recommendations with manual validation.

## Quick Start
1. Ensure Docker is running.
2. `docker-compose up -d`
3. `pip install -r requirements.txt`
4. `python backend_api/main.py`
5. Navigate to `unified_operator_interface` and run your dev server.
